⚡ AWS Infrastructure Automation

Net2Terraform

Generate production-grade AWS Terraform configurations from your network topology — choose the method that suits you best.

🖼️
📷 Vision · YOLO

Image Topology

Upload a scanned or drawn network diagram and let the AI automatically detect devices and connections, then generate a ready-to-deploy Terraform configuration.

  • YOLO-based device detection
  • Automatic topology graph extraction
  • Visual annotation with bounding boxes
  • OCR-based device name recognition
  • One-click AWS deployment via Terraform
OR
💬
🤖 RAG · LLM

Describe Your Network

Describe your network architecture in plain language through an interactive AI chat. The assistant will ask clarifying questions, validate the topology, and generate Terraform code.

  • Natural language architecture input
  • Retrieval-Augmented Generation (RAG)
  • Iterative clarification dialogue
  • Automatic VLSM IP addressing
  • Rules-aware Terraform generation

✅ Quality Assurance

Test & Evaluation

Run comprehensive test cases to validate the RAG and LLM result quality. Evaluate Terraform generation against expected architecture properties.

  • Pre-defined test cases
  • RAG accuracy metrics
  • Connectivity mode validation
  • Terraform quality checks
  • Pass/fail reporting
